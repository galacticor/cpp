This is the web game of the IEEEXtreme 14.0 challenge - The Last of Us.
Please read the problem statement for the detailed description of the game rules and task.

To start the web game, open "index.html" in a web browser.
The game requires JavaScript. Please enable JavaScript in your browser.

In the web game,
- Use the dropdown menu to select an area, then click "Start" to start playing.
- Click on a highlighted blue cell to walk into it.
- Click the "Throw Brick" button to switch to throwing bricks. Click on a highlighted blue cell to select the target cell to throw the brick at.
- Alerted clickers will be shown in red.
